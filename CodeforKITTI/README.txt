######Stereo Matching Code for KITTI Dataset#######

How to set up - 

1) Install the following libraries :- 
	i) opencv3.0 or greater
	ii) install opencv along with the opencv_contrib libraries
	iii)install boost libraries


2) Give the path for KITTI dataset in example.cpp
3) Provide the corresponding fileType



Steps to run the code :- 

1) Open a terminal 
2) cd ../CodeforKITTI/
3) mkdir build
4) cd build/
5) cmake ..
6) make
7) cd ..
5) cd bin/
6) ./example
7) plot.m contains the output for all census window types.
